방덱 목록 관리 도움말

기본 기능

  • 새 목록 추가: 상단의 입력창에 제목을 입력하고 Enter를 누르세요.
  • 목록 편집: 목록을 클릭하여 메모를 추가하거나 편집할 수 있습니다.
  • 목록 삭제: 목록 우측의 X 버튼을 클릭하세요.

메모 기능

  • 메모 추가: 목록을 클릭하고 메모 입력창에 내용을 입력한 후 Enter를 누르세요.
  • 메모 편집: 메모를 클릭하여 내용을 수정할 수 있습니다.
  • 메모 삭제: 메모 우측의 X 버튼을 클릭하세요.

클립보드 기능

  • 클립보드 토글: 상단의 클립보드 버튼을 클릭하여 클립보드를 열고 닫을 수 있습니다.
  • 항목 추가: 클립보드가 열린 상태에서 + 버튼을 클릭하여 새 항목을 추가할 수 있습니다.
  • 단축키: Alt + 1~9 키를 눌러 해당 위치의 클립보드 항목을 입력할 수 있습니다.
  • 최대 9개까지 항목을 저장할 수 있습니다.

데이터 저장

  • 모든 데이터는 자동으로 저장되며, 브라우저를 닫았다가 다시 열어도 유지됩니다.
  • 목록과 메모는 Firebase에 저장되며, 인터넷 연결이 없는 경우 로컬 저장소에 임시 저장됩니다.
