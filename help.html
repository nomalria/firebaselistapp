방덱 목록 관리 사용 가이드

⚠️ 주의사항 ⚠️ PC에서 사용하는 것을 전제로 웹이 구성되어 있습니다. 모바일 환경에서는 일부 기능이 제한될 수 있습니다.

1. 기본 기능 안내

  • 이 웹은 CURSOR AI를 통해 만들어졌습니다.
  • 임시 목록: 임시로 저장하는 목록으로, 나중에 기존 목록으로 이동할 수 있습니다.
  • 기존 목록: 목록이 일차적으로 저장되는 장소입니다.

2. 목록 추가 및 검색

상단의 검색창을 통해 기존 목록을 검색하거나 새 목록을 추가할 수 있습니다.

  • 몹 이름 형식: 목록의 몹들은 비 각성형의 이름으로 저장되어 있으며 오리지널 몹의 경우 콜라보 몹의 이름으로 되어있습니다.
  • 새 목록 추가: 검색창에 새 목록 이름을 입력하고 '추가' 버튼을 클릭하거나 Enter 키를 누르면 새 목록이 생성됩니다.
  • 검색 기능: 검색창에 입력한 단어가 3단어 이하일 경우 해당 단어를 포함하는 목록들을 임시목록으로 옮깁니다.
  • 단어 구분: 단어의 기준은 공백을 기준으로 판별합니다.
  • 추천 단어 선택: 검색창에 단어를 입력하는 동안 관련 추천 단어가 표시됩니다. 키보드 ↑, ↓ 화살표 키로 추천 단어 사이를 이동할 수 있으며, 스페이스바를 눌러 현재 선택된 추천 단어를 자동으로 입력할 수 있습니다.

💡 팁: 검색 결과에 해당하는 목록이 없으면 자동으로 새 목록으로 추가됩니다.

💡 팁: 추천 단어 기능을 사용하면 기존에 입력했던 몬스터의 이름을 더 빠르게 입력할 수 있습니다.

3. 목록 필터링

카테고리 버튼을 사용하여 목록을 필터링할 수 있습니다.

  • 전체보기: 모든 목록을 표시합니다.
  • 4방덱: 4방덱으로 분류된 목록만 표시합니다.
  • 5방덱: 5방덱으로 분류된 목록만 표시합니다.
  • 기타: 기타 카테고리로 분류된 목록만 표시합니다.

4. 클립보드 기능

자주 사용하는 메모를 클립보드에 저장하고 단축키로 빠르게 입력할 수 있습니다.

  • 클립보드 등록: 하단의 '클립보드 설정'에서 자주 사용하는 메모를 등록할 수 있습니다.
  • 단축키 사용: 메모 입력 중 Alt + 숫자키(1~9)를 누르면 해당 위치에 저장된 클립보드 내용이 입력됩니다.
  • 저장 개수: 최대 9개까지 클립보드 항목을 저장할 수 있습니다.

💡 팁: 클립보드 내용은 기기에 저장되므로 동일한 기기에서 계속 사용할 수 있습니다.

5. 메모 승/패 기록 관리

각 메모에 승리와 패배 횟수를 기록하고 관리할 수 있습니다.

  • 승리 카운터: 메모 오른쪽의 + 버튼으로 승리 횟수를 증가시킬 수 있습니다.
  • 패배 카운터: 메모 오른쪽의 - 버튼으로 패배 횟수를 증가시킬 수 있습니다.
  • 승패 아이콘: 승/패 비율에 따라 메모의 상태 아이콘이 자동으로 변경됩니다:
    • 승률 60% 이상: (적정)
    • 승률 41~59%: (보통)
    • 승률 40% 미만: (위험)

6. 데이터 관리

목록 데이터를 관리할 수 있는 기능을 제공합니다.

  • JSON 불러오기: 이전에 내보낸 JSON 파일을 불러와 목록을 복원할 수 있습니다.
  • JSON 내보내기: 현재 목록을 JSON 파일로 내보내 백업할 수 있습니다.
  • 목록 정렬: 목록과 메모를 알파벳순으로 정렬할 수 있습니다.
  • 임시목록 이동: 임시 목록을 기존 목록으로 이동할 수 있습니다.

💡 팁: 정기적으로 JSON 내보내기를 통해 데이터를 백업하는 것이 좋습니다.

7. 로그인 및 데이터 동기화

Firebase를 통해 데이터를 클라우드에 저장하고 여러 기기에서 동기화할 수 있습니다.

  • 로그인 방법: 우측 상단의 '로그인하기' 버튼을 클릭하여 Google 계정으로 로그인할 수 있습니다.
  • 자동 저장: 로그인하면 모든 목록 데이터가 클라우드에 자동 저장되고 동기화됩니다.
  • 기기간 동기화: 다른 기기에서 동일한 계정으로 로그인하면 목록 데이터가 자동으로 동기화됩니다.

💡 팁: 오프라인 상태에서는 로컬 저장소에 데이터가 저장되지만, 기기를 변경하거나 브라우저 데이터를 삭제하면 데이터가 손실될 수 있습니다.

8. 목록 및 메모 관리

목록과 메모를 편집, 삭제, 관리할 수 있는 다양한 기능을 제공합니다.

  • 메모 추가: 각 목록 하단의 입력창에 메모를 입력하고 Enter 키를 누르거나 '추가' 버튼을 클릭하여 메모를 추가할 수 있습니다.
  • 편집 기능: 목록 제목이나 메모 옆의 편집 버튼을 클릭하여 내용을 수정할 수 있습니다.
  • 삭제 기능: 목록 제목이나 메모 옆의 삭제 버튼을 클릭하여 삭제할 수 있습니다. 삭제 버튼을 두 번 클릭해야 최종 삭제됩니다.
  • 삭제 제한: 메모의 경우 삭제 제한이 없지만 이전 목록의 경우 삭제에 제한이 있습니다.
  • 메모 댓글: 메모에 댓글을 추가하여 추가 정보를 기록할 수 있습니다.

9. 페이지네이션

많은 수의 목록을 효율적으로 관리하기 위해 페이지네이션 기능을 제공합니다.

  • 항목 표시: 한 페이지에 최대 20개의 목록이 표시됩니다.
  • 페이지 이동: 목록 하단의 페이지 번호를 클릭하여 다른 페이지로 이동할 수 있습니다.
