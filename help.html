방덱 목록 관리 도움말

검색 및 추가 기능

  • 단어 검색

    입력창에 단어를 입력하면 실시간으로 추천 단어가 표시됩니다.

  • 추천 단어 선택
    • PC 사용 시:
      • 스페이스바 - 첫 번째 추천 단어 선택
      • ↑ ↓ - 추천 단어 이동
      • Tab - 다음 추천 단어로 이동
      • Shift + Tab - 이전 추천 단어로 이동
      • Enter - 선택된 추천 단어 입력
    • 모바일 사용 시:
      • 스페이스 입력 또는 추천 단어 직접 터치
  • 단어 분리 검색

    목록의 각 단어가 개별적으로 검색됩니다.

    예시) "4방덱 물무희" 목록이 있을 때:
    • "4" 입력 → "4방덱" 추천
    • "물" 입력 → "물무희" 추천
  • 임시 목록 자동 이동

    3개 이하의 단어를 입력하면 다음과 같이 처리됩니다:

    • 기존 목록과 일치할 경우: 해당 목록이 임시 목록으로 자동 이동
    • 일치하는 목록이 없는 경우: 새로운 임시 목록 생성
    예시) "물무희" 입력 시:
    • 기존 목록에 "4방덱 물무희"가 있다면 → 임시 목록으로 이동
    • 일치하는 목록이 없다면 → 새로운 임시 목록 생성

기본 기능

  • 새 목록 추가

    상단의 입력창에 제목을 입력하고 Enter를 누르세요.

  • 목록 편집

    목록을 클릭하여 메모를 추가하거나 편집할 수 있습니다.

  • 목록 삭제

    목록 우측의 X 버튼을 클릭하세요.

메모 기능

  • 메모 추가

    목록을 클릭하고 메모 입력창에 내용을 입력한 후 Enter를 누르세요.

  • 메모 편집

    메모를 클릭하여 내용을 수정할 수 있습니다.

  • 메모 삭제

    메모 우측의 X 버튼을 클릭하세요.

  • 메모 상태 설정
    • 상태 아이콘:
      • ✅ - 공격 성공
      • ❌ - 공격 실패
    • 설정 방법:
      • 단축어 사용:
        • 성공: 공성 또는 공격성공
        • 실패: 공실 또는 공격실패
        • (단축어 사용시 해당 메모의 맨 끝에 입력하셔야합니다, 단축어 이후의 메모 내용은 모두 삭제됩니다.)
      • 아이콘 클릭: 메모 우측의 ✅ 또는 ❌ 버튼 클릭

클립보드 기능

  • 클립보드 열기/닫기

    상단의 클립보드 버튼을 클릭하여 클립보드를 열고 닫을 수 있습니다.

  • 항목 추가

    클립보드가 열린 상태에서 + 버튼을 클릭하여 새 항목을 추가할 수 있습니다.

  • 단축키

    Alt + 1~9 키를 눌러 해당 위치의 클립보드 항목을 입력할 수 있습니다.

  • 저장 제한

    최대 9개까지 항목을 저장할 수 있습니다.

데이터 저장

  • 자동 저장

    모든 데이터는 자동으로 저장되며, 브라우저를 닫았다가 다시 열어도 유지됩니다.

  • 저장 위치

    목록과 메모는 Firebase에 저장되며, 인터넷 연결이 없는 경우 로컬 저장소에 임시 저장됩니다.
